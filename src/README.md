# Web3 Snake

Minimal demo: connect your MetaMask wallet and play a snake game.

## How to run locally

1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

## Deploy to Vercel

1. Create a GitHub repo and push this project.
2. Go to Vercel, import project from GitHub, pick the repo.
3. Build command: `npm run build`. Output directory: `dist`.
4. Deploy.
